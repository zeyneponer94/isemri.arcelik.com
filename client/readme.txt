packages/: folder contains the BotDetect Java Captcha library redistributable packages and their javadoc documentation.

examples/: folder contains BotDetect Java Captcha library code examples and demos. Deploy .war files to your servlet container (copy to autodeploy folder or use admin/manager console) and point your browser to index (e.g. http://localhost:8080/bdc4-captcha-index) to run examples.

changelog.txt: file contains the version history of BotDetect Java Captcha library releases.

botdetect-license.html: file contains the licensing terms for using BotDetect Java Captcha library.
